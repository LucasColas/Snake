# Snake
Snake made with Python and Pygame !

## Prerequisites 
* [Python >= 3.10](https://www.python.org/)

* Pygame or [pygame-ce](https://pyga.me/) (I used pygame-ce)

## How to play
* git clone this repository
  ```bash
  git clone git@github.com:LucasColas/Snake.git
  ```
* go to the directory
  ```bash
  cd Snake
  ```
* run `main.py`
* Enjoy !
